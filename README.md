# Goldenmont Academy

A short browser-based school-life visual novel with stat raising, two male leads, hidden relationship values, generated 128×128 pixel art, three endings, and local save/load.

## Play

Open the GitHub Pages site or run `index.html` locally in a modern browser.

## Prototype scope

- 14 in-game days
- Female customizable protagonist
- Five stats: Academics, Charisma, Creativity, Fitness, Kindness
- One activity per day
- Childhood-friend and popular-student routes
- Hidden affection with relationship hints
- 128×128 procedural pixel portraits and school backgrounds
- Browser local-storage save/load

No build process or external dependencies are required.
